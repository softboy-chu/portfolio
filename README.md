# Okeke Gabriel Chukwuma — Portfolio

Personal portfolio site. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step, no dependencies.

**Live:** [)

---

## Stack

| Layer | Choice |
|---|---|
| Markup | HTML5 |
| Styles | CSS3 (custom properties, CSS Grid, Flexbox) |
| Scripts | Vanilla JavaScript |
| Fonts | Cormorant Garamond + DM Sans (Google Fonts) |
| Hosting | Netlify |

---

## Structure

```
portfolio/
├── index.html   ← all sections on one scrollable page
├── style.css    ← all styles, variables, and responsive breakpoints
└── app.js       ← mobile menu toggle and scroll lock
```

---

## Sections

- **Hero** — headline, tagline, CTA
- **Work** — selected projects (Admin Dashboard, Haven E-commerce, Nova UI Kit)
- **About** — bio, stats, tech stack
- **Contact** — email, socials, footer
